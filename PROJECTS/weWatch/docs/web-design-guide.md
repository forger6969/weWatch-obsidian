## Цветовая палитра

### Фон (от тёмного к светлому)
| Название | Hex | Использование |
|----------|-----|--------------|
| `bg-void` | `#060608` | Root фон |
| `bg-base` | `#0A0A0F` | Страница, inputs |
| `bg-elevated` | `#111118` | Card, modal |
| `bg-overlay` | `#16161F` | Hover overlay |
| `bg-surface` | `#1C1C28` | Secondary card |

### Brand Colors
| | Hex |
|-|-----|
| Primary (violet) | `#7B72F8` |
| Hover | `#6B63E8` |
| Gold (achievement) | `#FFD700` |
| Diamond (top rank) | `#88CCFF` |
