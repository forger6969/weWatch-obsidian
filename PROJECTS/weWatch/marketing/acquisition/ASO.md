# 📱 ASO — App Store Optimization

## App Store (iOS)

### Название (30 символов)
```
WeWatch — Watch Together
```

### Подзаголовок (30 символов)
```
Кино с друзьями на расстоянии
```

### Ключевые слова (100 символов)
```
watch party,together,movie,sync,friends,couple,long distance,film,кино,вместе
```

### Описание (короткое — первые 3 строки КРИТИЧНЫ)
```
Смотри фильмы вместе с теми, кто далеко.

Создай комнату → пошли ссылку → через секунду вы вместе.
Синхронная пауза, общий чат, реакции. Без лагов. Без настройки.

WeWatch — для тех, кто хочет быть рядом несмотря на расстояние.
```

### Скриншоты (порядок важен)
1. **Room с фильмом** — главный экран просмотра (эмоция)
2. **Share link** — "Позови друга одним тапом"
3. **Чат + реакции** — общение во время фильма
4. **Battle** — "Кто посмотрел больше за неделю?"
5. **Achievements** — бейджи и достижения

### Иконка
- Тёмный фон (#0A0A0F)
- Фиолетовый акцент (#7B72F8)
- Символ: два силуэта + экран или play button
- Минимализм, читается в 29px

## Отзывы и рейтинг

**Стратегия сбора отзывов:**
- In-app prompt: после 3-й сессии и после оценки фильма
- Никогда не просить сразу после установки
- Тайминг: конец хорошей сессии (оценка ★★★★★ в end-of-session modal)

**Цель:** > 50 отзывов за первый месяц, рейтинг > 4.3
