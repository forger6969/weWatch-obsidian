# 🔗 Viral Loop — WeWatch

## Главная механика

```
Пользователь А создаёт room
          ↓
Шарит ссылку (WhatsApp / Telegram / SMS)
          ↓
Пользователь Б нажимает ссылку
          ↓
    ┌─────────────────────────────────┐
    │  Есть приложение WeWatch?       │
    │  ДА  → deep link → в room       │  ← retention
    │  НЕТ → web room (браузер)       │  ← acquisition
    └─────────────────────────────────┘
          ↓
  Конец сессии → End-of-session modal
    "Создавай комнаты сам → Скачай приложение"
          ↓
Пользователь Б скачивает → становится А
          ↓
          LOOP
```

---

## Viral Coefficient (K-factor)

```
K = i × c

i = среднее количество приглашений на пользователя
c = конверсия (% нажавших ссылку)

Цель Фазы 1: K = 0.5 (каждые 2 пользователя = 1 новый)
Цель Фазы 2: K = 1.0+ (органический рост без бюджета)
```

**Как увеличить K:**
- Увеличить `i`: реферальная программа, "пригласи друга" CTA
- Увеличить `c`: красивый rich preview, web room без регистрации

## Точки усиления вирусности

### 1. Rich Link Preview *(нужно сделать)*
Когда ссылка отправляется в WhatsApp/Telegram:
```
┌────────────────────────────┐
│ 🎬 [Постер фильма]         │
│ Лейла зовёт тебя в кино    │
│ "Дюна 2" — смотрим вместе  │
│ Нажми, чтобы присоединиться│
└────────────────────────────┘
```
**Эффект:** CTR × 2–3

### 2. Web Room без регистрации *(нужно сделать)*
- Нажал ссылку → сразу в комнате
- Регистрация — только если хочет создавать room
- **Эффект:** конверсия × 2–3 (без трения)

### 3. End-of-Session Modal *(обсуждалось)*
После окончания фильма:
```
Как вам вечер? ★★★★★
[Оставить отзыв]
         ↓
Создавай комнаты сам — скачай WeWatch!
[App Store]  [Google Play]
```
**Эффект:** конверсия web → app

### 4. Реферальная программа *(Фаза 2)*
- Пригласил друга → он зарегистрировался → ты получаешь бейдж "Connector"
- 5 приглашений → "Social Butterfly" achievement
- **Эффект:** i × 1.5–2

### 5. "Ждём тебя в room" push *(нужно сделать)*
Когда друг создал room и добавил тебя:
- Push: "Акбар ждёт тебя в кино 🎬"
- **Эффект:** конверсия invite → join × 1.5

## Метрики вирусности (трекать еженедельно)

| Метрика | Формула | Цель |
|---------|---------|------|
| K-factor | invites/user × conversion | > 1.0 |
| Invite Rate | % пользователей отправивших ссылку | > 60% |
| Join Rate | % нажавших ссылку → зашли в room | > 40% |
| Install Rate | % web users → скачали приложение | > 15% |
| Time to Viral | дней от регистрации до первого invite | < 1 день |
