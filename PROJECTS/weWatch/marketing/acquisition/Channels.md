## Канал 1 — Share Link (Viral Loop) ⭐

**Статус:** ✅ Реализован в приложении

**Механика:**
- Пользователь создаёт room → копирует ссылку
- Друг нажимает → есть приложение = deep link в room
- Нет приложения = web room → видит продукт → скачивает

**Улучшения:**
- [ ] OG image с постером фильма (rich preview в чатах)
- [ ] Web room с CTA "Скачай приложение — создавай комнаты сам"
- [ ] End-of-session modal: оценка + download CTA
