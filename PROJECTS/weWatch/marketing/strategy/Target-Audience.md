# 🎯 Target Audience — WeWatch

> Мы не продаём стриминг. Мы продаём **ощущение присутствия**.

---

## Сегмент 1 — Парочки на расстоянии ⭐ ГЛАВНЫЙ

| Параметр | Описание |
|---------|---------|
| **Возраст** | 18–28 |
| **Ситуация** | Один учится/работает в другом городе или стране |
| **Боль** | "Мы не можем просто лечь на диван и смотреть кино вместе" |
| **Частота использования** | 2–3 раза в неделю |
| **Где найти** | Instagram (travel couples), TikTok #ldr, студ. чаты |
| **Триггер-момент** | Сентябрь (начало учебного года — разлука) |
| **Послание** | *"Диван на двоих — без расстояний"* |

**Почему главный:** Самая высокая частота, самая острая боль, самый высокий LTV.

---

## Сегмент 2 — Друзья в разных городах

| Параметр | Описание |
|---------|---------|
| **Возраст** | 17–25 |
| **Ситуация** | Переехали учиться/работать, потеряли совместный ритуал |
| **Боль** | "Мы раньше всегда смотрели по пятницам — теперь нет" |
| **Частота использования** | 1 раз в неделю (пятница/суббота вечер) |
| **Где найти** | Telegram, Discord, студенческие чаты |
| **Триггер-момент** | "Смотришь что-нибудь сегодня?" |
| **Послание** | *"Пятничный кинотеатр — где бы вы ни были"* |

## Сегмент 3 — Семья (дети и родители)

| Параметр | Описание |
|---------|---------|
| **Возраст** | 22–35 (дети) + 45+ (родители) |
| **Ситуация** | Дети уехали учиться/работать за рубеж |
| **Боль** | "Хочу провести время с мамой, но она в другом городе" |
| **Частота использования** | 1 раз в 1–2 недели |
| **Где найти** | Facebook, WhatsApp семейные группы |
| **Послание** | *"Семейный вечер без километров"* |

## Анти-аудитория (не наши)

- Одиночные зрители → им нужен Netflix/YouTube
- Люди которые живут рядом → им не нужен WeWatch
- 40+ без опыта мобильных приложений
- Люди без стабильного интернета

## Ключевая персона — Лейла и Акбар

> **Лейла** (21 год, студентка, Ташкент)
> **Акбар** (23 года, программист, Москва)
>
> Каждую пятницу в 21:00 они "идут в кино".
> Акбар создаёт room, шарит ссылку в WhatsApp.
> Лейла нажимает — они вместе смотрят фильм.
> После — ставят оценку, обсуждают в чате.
> Следующая пятница: "Дюна 3?"
>
> **Это наш core user. Всё делаем для них.**

## Психографика (что им важно)

- Близость > качество видео
- Простота > функциональность
- Ритуал > разовый опыт
- "Наш app" > безликий инструмент
